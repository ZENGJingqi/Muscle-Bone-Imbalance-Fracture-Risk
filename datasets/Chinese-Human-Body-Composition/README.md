# Chinese Human Body Composition Dataset

## Role in the Study

This dataset corresponds to the original Chinese discovery cohort used in the manuscript. It provided the initial `BIA-based MBR` signal and the original cohort-specific threshold explored in the paper.

This repository does **not** redistribute the original Chinese participant-level data.

## Availability of Data and Material

The **"Human Body Composition Dataset for the Chinese Population"** can be accessed through the National Population Health Data Center:

- main portal: <https://www.ncmi.cn/>
- dataset page: <https://www.ncmi.cn//phda/dataDetails.do?id=CSTR:A0006.11.A0005.201905.000346>

License:

## Important Note

This repository is mainly intended to document:

1. the study context,
2. the external datasets used for extension and validation,
3. the lightweight public analysis workflow for those external datasets.

Because the Chinese cohort is not redistributed here, no public participant-level data files are included in this folder.

## Interpretation

The original Chinese dataset supports the discovery-stage `BIA-based MBR` finding. The external datasets in this repository were used to evaluate:

- cross-platform bridge,
- external DXA consistency,
- older-adult clinical outcome context.

Therefore, the original Chinese cutoff should be interpreted as a cohort-specific discovery threshold rather than a universally transferable screening cutoff.
